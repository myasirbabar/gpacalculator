# GPA AND CGPA CALCULATOR
For PUCIT Build using C++ console

------------------


### How to begin
- Open Up the main file in editor
- Compile And Run
- Happy Execution :)

### Functionalities
- Calculates GPA / CGPA of a PUCIT Student. 
- Geneates Formatted Transcript in .rtf extension for GPA / CGPA
